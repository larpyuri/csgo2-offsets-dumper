# Overview

csgo2-offsets provides automated offset dumping and structured output generation to assist with 
reverse engineering research, tooling development, and technical analysis related to Counter-Strike 2.

# Usage

Launch Counter-Strike 2.

Ensure the game is fully loaded.

Run csgo2-dumper.exe with administrator privileges.

The tool will automatically scan and generate the output files.

# Requirements

Windows 10 or Windows 11

Administrator privileges

Running instance of Counter-Strike 2
